## **STEP 1: COPY TO USB** 📁
1. **Copy entire `DreamletEduVideo` folder to USB drive**
2. **That's all you need to copy!**

---

## **STEP 2: ON ANY COMPUTER** 💻

### **🪟 Windows:**
1. **Insert USB drive**
2. **Double-click: `START_HERE.bat`**
3. **Wait 5 minutes** ⏳
4. **Browser opens automatically** 🌐
5. **Done!** ✅

### **🍎 Mac / 🐧 Linux:**
1. **Insert USB drive** 
2. **Double-click: `START_HERE.sh`** (or run in Terminal)
3. **Wait 5 minutes** ⏳
4. **Browser opens automatically** 🌐
5. **Done!** ✅

---

### **Option A: Use Desktop Folders** 📂
- **Put files in:** `dreamlet_input` folder (appears on Desktop)

## **❓ TROUBLESHOOTING:**

**Problem:** Script won't run  
**Solution:** Right-click → "Run as Administrator" (Windows) or use Terminal (Mac/Linux)

**Problem:** Docker error  
**Solution:** Install Docker Desktop from docker.com first

**Problem:** Browser doesn't open  
**Solution:** Manually go to http://localhost:8501

**Problem:** Files not processing  
**Solution:** Check input folder has correct file types (.txt, .md, .pptx)
