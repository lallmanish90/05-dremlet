
---

## **STEP 2: COPY EVERYTHING** 📋

1. **Copy entire `DreamletEduVideo` folder to USB drive**
2. **That's it!** - Everything needed is included

---

## **STEP 3: ON ANY COMPUTER** 💻

### **Windows Users:**
1. **Insert USB drive**
2. **Double-click `START_HERE.bat`** 
3. **Wait 5 minutes** (downloads Docker, sets everything up)
4. **Browser opens automatically** to `http://localhost:8501`
5. **Done!** 🎉

### **Mac/Linux Users:**
1. **Insert USB drive**
2. **Open Terminal**
3. **Navigate to USB folder:**
   ```bash
   cd /path/to/usb/DreamletEduVideo
   ```
4. **Run setup:**
   ```bash
   chmod +x START_HERE.sh && ./START_HERE.sh
   ```
5. **Browser opens automatically** to `http://localhost:8501`
6. **Done!** 🎉

---

## **STEP 4: USE THE APPLICATION** 🎯

### **Put files in:**
- Look for `dreamlet_input` folder on desktop
- Drop your educational files there

### **Get results from:**
- Look for `dreamlet_output` folder on desktop
- Your videos appear there

### **OR use web interface:**
- Click "File Manager" in the app
- Upload files directly through browser
- Download results as ZIP files
