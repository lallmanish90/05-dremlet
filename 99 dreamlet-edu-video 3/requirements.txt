# Installation Instructions:

# 1. Basic installation:

# 2. For GPU support (NVIDIA):
#    pip install onnxruntime-gpu==1.16.3 --upgrade
#
# 3. Install FFmpeg (system dependency):
#    macOS: brew install ffmpeg
#    Ubuntu: sudo apt-get install ffmpeg
#    Windows: Download from https://ffmpeg.org/download.html
#
# 4. For Kokoro TTS models, download from:
#    https://github.com/kokoro-tts/models
#
# 5. For development:
#    pip install pytest black pylint mypy

# ===========================================
# System Requirements:
# ===========================================
# - Python 3.8 or higher
# - FFmpeg installed on system
# - 8GB+ RAM recommended
# - GPU with CUDA support (optional, for acceleration)
# - 10GB+ free disk space for models and processing
