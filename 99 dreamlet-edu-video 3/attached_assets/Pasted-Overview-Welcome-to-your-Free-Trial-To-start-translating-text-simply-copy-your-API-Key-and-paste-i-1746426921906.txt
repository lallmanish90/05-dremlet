Lecto Translation API (1.0.0)
Download OpenAPI specification:Download

Lecto Translation API provides API endpoints for translation between 90+ source and target language pairs with multiple target languages supported in the same request.

API Key
This API uses API Key for authentication. This API key must be sent as the value of the header X-API-Key. Sign up for a Free Translation API trial to obtain your API Key which is authorized to access the Lecto Translation API.

After signing in, your API Key will appear here:

Compression
This API supports gzip compressed responses to save bandwidth on your client. Send the optional Accept-Encoding: gzip header to receive gzip compressed responses.

If this header is sent, then the API response will be gzip compressed and will include the Content-Encoding: gzip response header to indicate gzip compression.

JSON
Translate JSON payload
Send source JSON in request body and receive JSON response with translations in multiple target languages. JSON values are translated while keys are left unchanged.

Authorizations:
ApiKeyAuth
Request Body schema: application/json
json data request body

from
required
string
Source language code to translate from. Source Language code is required for JSON payload translations.

json
required
string
Original JSON formatted string to translate from. Use the JSON.stringify Javascript method or similar to generate this JSON formatted string from your source JSON payload.

protected_keys	
Array of strings
Array of JSON Keys for which translation is to be skipped.

to
required
Array of strings
Array of target language codes to translate into.

Responses
200 json translation result
400 bad request
401 unauthorized
404 not found
429 too many requests
