## Migration Checklist
1. Update all API endpoints from `127.0.0.1:1234` to `localhost:11434`
2. Replace LM Studio model references with Ollama model names
3. Add Ollama model verification checks
4. Update error messages and documentation references
5. Test with Ollama's streaming API
6. Adjust rate limiting parameters for local inference
7. Update system prompts for Ollama-optimized models
8. Verify context window sizes match Ollama model capabilities

Key differences to consider:
- Ollama uses port 11434 instead of LM Studio's 1234
- Model loading happens through `ollama pull` instead of LM Studio's GUI
- Ollama provides better GPU memory management
- Different default temperature settings may be needed
- Context window sizes vary between models

Remember to:
1. Install Ollama (https://ollama.ai/)
2. Pull required models (`ollama pull llama3`)
3. Start Ollama service (`ollama serve`)
4. Verify API access (`curl http://localhost:11434/v1/models`)
