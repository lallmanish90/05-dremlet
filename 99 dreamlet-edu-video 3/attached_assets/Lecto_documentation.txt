end-to-end reference of the Lecto Translation API :

## Overview  
Lecto Translation API v1.0.0 lets you detect languages and translate text or JSON between 90+ languages.

## Base URL  
All endpoints are rooted at:  
```
https://api.lecto.ai/v1
```

## Authentication  
All requests require an API key in the header:  
-  Header Name: `X-API-Key`  
-  Value: your case-sensitive API key (e.g. `P66PARQ-MH44AC6-HNA9BE2-25YRH37`)

## Common Headers  
-  `Content-Type: application/json`  
-  `Accept: application/json`  
-  Optional for response compression: `Accept-Encoding: gzip`

## Endpoints

### 1. Get Account Information  
Retrieves your current plan, usage and limits.  
- Method: GET  
- Path: `/account/info`  
- Auth: ApiKeyAuth  

Response (200 OK):  
```json
{
  "current_period_start": "2022-02-03T03:24:36Z",
  "current_period_end":   "2022-03-03T03:24:36Z",
  "plan":                "BUSINESS",
  "plan_status":         "active",
  "usage":               193729227,
  "usage_limit":         750000000
}
```
Errors: 400, 401, 404, 429

### 2. Detect Language of Text  
Auto-detects the source language for one or more text strings.  
- Method: POST  
- Path: `/detect/text`  
- Body Schema:  
  ```json
  {
    "texts": ["Hello World", "Bonjour le monde"]
  }
  ```
Response (200 OK):  
```json
{
  "detected_languages": ["en", "fr"],
  "original_characters": 23
}
```
Errors: 400, 401, 404, 429

### 3. List Supported Languages  
Returns all available language codes and names.  
- Method: GET  
- Path: `/translate/languages`  
Response (200 OK):  
```json
{
  "languages": [
    { "code": "en",   "name": "English" },
    { "code": "pt-BR","name": "Portuguese (Brazil)" },
    …
  ]
}
```
Errors: 400, 401, 404, 429

### 4. Translate Text or HTML  
Translates an array of texts into one or more target languages.  
- Method: POST  
- Path: `/translate/text`  
- Body Schema:  
  ```json
  {
    "from": "en",                  // optional; auto-detected if omitted
    "texts": ["Hello","Goodbye"],  // required
    "to": ["de","es"]              // required
  }
  ```
Response (200 OK):  
```json
{
  "from": "en",
  "translated_characters": 24,
  "translations": [
    { "to": "de", "translations": ["Hallo","Auf Wiedersehen"] },
    { "to": "es", "translations": ["Hola","Adiós"] }
  ]
}
```
Errors: 400, 401, 404, 429

### 5. Translate JSON Payload  
Translates string values in a JSON document; keys remain unchanged.  
- Method: POST  
- Path: `/translate/json`  
- Body Schema:  
  ```json
  {
    "from": "en",                              // required
    "json": "{\"greeting\":\"Hello\",\"name\":\"World\"}",  
    "protected_keys": ["name"],                // optional
    "to": ["fr","jp"]                          // required
  }
  ```
Response (200 OK):  
```json
{
  "from": "en",
  "protected_keys": ["name"],
  "translated_characters": 11,
  "translations": [
    { "to": "fr", "json": "{\"greeting\":\"Bonjour\",\"name\":\"World\"}" },
    { "to": "jp", "json": "{\"greeting\":\"こんにちは\",\"name\":\"World\"}" }
  ]
}
```
Errors: 400, 401, 404, 429

## Language Codes  
Supported codes include ISO-639-1, ISO-639-2 and BCP-47. Examples:  
```
en, es, de, fr, zh-CN, zh-TW, pt-PT, pt-BR, ja, ko, ru, ar, hi,…
```

## Error Responses  
-  400 Bad Request – malformed input  
-  401 Unauthorized – missing/invalid API key  
-  404 Not Found – invalid endpoint  
-  429 Too Many Requests – rate limit exceeded
