Standard Structure: The system expects 3 types of files per lecture:

Files to Delete: Any file without a clear lecture number or with unsupported types (outlines, hybrid transcript-summaries, artifacts without numbers, duplicates with "(1)", "(2)" indicators) should be deleted.

Renaming Patterns:

artifact_X_transcript → Lecture X
artifact_X_summary → X-summary
slide_content_X → X-slides
video_transcript X → Lecture X
